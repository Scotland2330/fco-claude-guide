📁 How to use this repo with Claude Projects

  1. Click Download All Files above
  2. Go to github.com/new and create a private repo called my-claude-context
  3. Upload all the downloaded files to the repo root
  4. In Claude, open Projects and create a new project
  5. Click Add Content → Connect GitHub and select your repo
  6. Start every session with: "Read all attached files before responding. Ask clarifying questions before starting any task."
